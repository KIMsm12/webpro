러버블에게 요구한 프롬프트

"랜덤 면접 질문 생성 웹사이트"를 HTML/CSS/JavaScript로 제작해줘.

✅ 기능 요구
1) 버튼을 누르면 랜덤 면접 질문 1개가 화면에 표시된다.
2) 화면에 표시된 질문은 계속 변경 가능하다.
3) 유명한 면접 질문(인성 + 기술 + 상황 면접)들을 최소 50개 이상 포함해라.
4) 질문 데이터는 자바스크립트 배열로 저장해라.
5) 질문은 한국어로 표시한다.

✅ UI / 디자인 요구
1) 깔끔하고 모던한 디자인
2) 가운데 정렬
3) 부드러운 애니메이션 효과 (페이드인 or 스무스 등장)
4) 폰트는 간결하고 현대적인 스타일을 사용
5) 배경은 심플한 그라디언트 or 은은한 색감
6) 질문 카드(compact box) 형태
7) 모바일 반응형 디자인 포함

✅ 구성 요구
1) 하나의 완성 HTML 파일과 CSS, JS 파일 구조로 작성
   - index.html
   - style.css
   - script.js
2) 직접 실행 가능한 완성 코드 형태
3) 한국어 주석 추가
4) 버튼 UI는 hover 효과 적용
5) favicon 없어도 됨

✅ 추가 기능
1) “다음 질문” 버튼
2) “질문 복사하기” 버튼 추가 (클립보드 복사)
3) 현재 몇 번째 질문이 나왔는지 표시(카운터)

가능하면 아래와 같은 섹션 배치를 따라라:
1) 제목 영역
2) 질문 카드
3) 버튼 2개 (다음 질문 / 질문 복사)
4) footer (간단한 문구)

전체적으로 세련된 면접 준비 서비스처럼 구성해줘.
